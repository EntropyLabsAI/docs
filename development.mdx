## Development Process

To contribute to Asteroid, follow these steps:

<Steps>
  <Step title="Clone the repository">
    Clone the Asteroid repository to your local machine:
    ```bash
    git clone https://github.com/asteroidai/asteroid.git
    ```
  </Step>
  <Step title="Make changes">
    Edit the relevant files in the frontend or backend as needed.
  </Step>
  <Step title="Test changes locally">
    Follow the "Getting Started" instructions to test your changes locally.
  </Step>
  <Step title="Submit a pull request">
    Once your changes are tested, submit a pull request for review.
  </Step>
</Steps>

<Card title="Learn More" icon="book-open">

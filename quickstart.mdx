<Steps>
  <Step title="Clone the repository">
    Clone the Sentinel repository to your local machine and navigate to the project directory:
    ```bash
    git clone https://github.com/EntropyLabsAI/sentinel.git && cd sentinel
    ```
  </Step>
  <Step title="Set up environment">
    Copy the example environment file and set the variables:
    ```bash
    cp .env.example .env
    ```
    Edit the `.env` file to set the appropriate environment variables.
  </Step>
  <Step title="Load environment variables">
    ```bash
    source .env
    ```
  </Step>
  <Step title="Start the server and frontend">
    Use Docker Compose to start both the server and frontend:
    ```bash
    docker compose up
    ```
  </Step>
</Steps>

## Examples

Sentinel can be used with any agent by sending reviews to the `/api/review/{supervisor}` endpoint and checking their status with the `/api/review/status` endpoint.

<AccordionGroup>
  <Accordion icon="terminal" title="Using curl">
    <Steps>
      <Step title="Send a review">
        Use the `/api/review/human` endpoint:
        ```bash
        curl -X POST http://localhost:8080/api/review/human \
             -H "Content-Type: application/json" \
             -d @examples/curl_example/payload.json
        ```
        You should see `{"id":<review-id>,"status":"queued"}` printed in the terminal.
      </Step>
      <Step title="Check review status">
        Use the `/api/review/status` endpoint:
        ```bash
        curl http://localhost:8080/api/review/status/<review-id>
        ```
        Replace `<review-id>` with the ID of the submitted review.
      </Step>
      <Step title="View and manage reviews">
        Navigate to `http://localhost:3000` to see submitted reviews and approve or reject them.
      </Step>
    </Steps>
  </Accordion>

    </Info>
  </Accordion>
